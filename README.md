# NBA-2011-Overspending-Analysis
This project uses data from HoopsHype, Kaggle, ProBasketballReference, and Spotrac to examine the relationship between National Basketbll Association (NBA) team payroll, salary cap spending, and on-court performance during the 2011–2012 NBA season using Tableau. The dashboard analyzes whether higher team expenditures translate into greater success by examining wins, scoring, and other team performance metrics.

The project demonstrates how sports analytics and data visualization can be used to identify trends between financial investment and competitive outcomes in professional basketball.

The project uses publicly available NBA data with key variables, including:
  - Team payroll data
  - NBA salary cap information
  - Team standings
  - Team statistics
  - Individual player salary data

This project explores the following question:
  - To what extent can increasing overall team spending improve success throughout an NBA season?

By comparing salaries and win totals across the NBA, this analysis provides insight into the role financial resources may play in NBA team performance.

This repository includes the Tableau dashboard used to construct the analysis, along with the initial datasets containing team and salary data from the 2011-2012 NBA season.
